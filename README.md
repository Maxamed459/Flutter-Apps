# 🦋 Flutter Practice Apps

This repository is my personal Flutter practice space.

It contains multiple apps that I build while learning, testing ideas, and improving my Flutter skills. Right now there are two apps in the repo, and I will keep adding more practice projects here as I grow.

## 📦 What is inside

- `sneaker_shop` - a sneaker shopping app project focused on a polished UI, navigation, and app structure.
- `todo_app` - a task management app project that uses local storage and common Flutter interaction patterns.

## 🛠️ Current apps

### 👟 Sneaker Shop

The `sneaker_shop` app is a practice project for building a shopping-style Flutter interface.

It currently uses:

- `DevicePreview` for testing layouts on different screen sizes
- a welcome screen and home page structure
- Flutter Material widgets for the app shell

This app helps me practice:

- UI layout and styling
- screen navigation
- reusable page structure
- responsive previewing during development

### ✅ Todo App

The `todo_app` project is a task list app built to practice local state and persistence.

It currently uses:

- `Hive` and `Hive Flutter` for local storage
- `flutter_slidable` for swipe actions
- `DevicePreview` for responsive testing

This app helps me practice:

- saving data locally
- task management flows
- list interactions
- Flutter state and UI updates

## 🚀 Future apps

This repository is intentionally open-ended.

As I continue learning Flutter, I will add more apps here to explore new ideas, packages, UI patterns, and app architectures. Each new app will live in its own folder so the repository becomes a growing collection of practice projects.
